### DS-GA 1018: Probabilistic Time Series Analysis

This repository contains notebooks that are worked out as part of lab sessions at NYU.

#### Overview

This repository serves as a comprehensive resource for DS-GA 1018: Probabilistic Time Series Analysis, a graduate-level course at Centre for Data Science, New York University. It contains curated Jupyter notebooks and supporting materials developed during weekly lab sessions, designed to reinforce core concepts and provide practical implementations of time series methodologies.

#### Contents

The notebooks cover fundamental and advanced topics including:
- Autoregressive (AR) Modeling
- Autoregressive Moving Average (ARMA) Modeling
- Discrete Hidden Markov Models
- Gaussian Distributions
- Gaussian Processes
- Kalman Filtering and Smoothing
- Latent Dynamical Systems and EM Algorithm
- Particle Filtering

#### Prerequisites

- Python 3.8 or higher
- NumPy, SciPy, Pandas, Matplotlib and Seaborn
- Jupyter Notebook or JupyterLab

#### Usage

Clone the repository and navigate to the desired notebook:

```bash
git clone <repository-url>
cd NYU_Probabilistic_Time_Series_Analysis
jupyter notebook
```
